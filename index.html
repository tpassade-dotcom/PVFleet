⚙ Configuration Proxy Cloudflare

Le proxy Cloudflare Worker est nécessaire pour accéder aux APIs SolarEdge et Solis depuis le navigateur.
Déploie cloudflare-worker.js sur Cloudflare Workers et colle l'URL ci-dessous.

Comment déployer :
1. Va sur dash.cloudflare.com → Workers & Pages → Create Worker
2. Colle le contenu de cloudflare-worker.js → Save & Deploy
3. Dans le worker, modifie ALLOWED_ORIGINS pour y ajouter ton domaine GitHub Pages
4. Copie l'URL générée et colle-la ici
